# tdp
Different simple examples for HPC : MPI, BLAS ...

## 3tdp

Fox Multiplication using MPI

## 5tdp-hk

Some BLAS/LAPACK implementations (cmake compatible with MKL and others):
- dscal
- dger
- dgemm
- dtrsm
- dgetf2
- dgetrf
- dgetrf (with MPI)
- dgesv

## 6tdp-hk

Life-game with different implementations:
- sequential
- OpenMP
- pthread
- MPI
